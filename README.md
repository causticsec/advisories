# Advisories

Published security advisories from Caustic.

## ID scheme

`CAUSTIC-YYYY-NNN` - year, then a zero-padded three-digit per-year sequence (e.g. `CAUSTIC-2026-001`).

## Layout

One folder per year:

```
2026/
```

## Process

Advisories follow coordinated disclosure: 90 days from vendor notification. Each advisory is published after a fix is available or at the 90-day deadline, whichever comes first. Active exploitation in the wild may shorten the timeline. See the full [Vulnerability Disclosure Policy](https://github.com/causticsec/.github/blob/main/VULNERABILITY_DISCLOSURE_POLICY.md).
